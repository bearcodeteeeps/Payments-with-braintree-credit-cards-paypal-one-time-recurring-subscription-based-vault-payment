Braintree Payments

Introduction
	Braintree, a division of PayPal, is a company based in Chicago that specializes in mobile and web payment systems for e-commerce companies that was acquired by PayPal on September 26, 2013. 

Steps For payments with braintree

	Website Url -: https://www.braintreepayments.com/

=> Look for login & then select signup for sandbox 01.png

=> Fill the form & submit.

=> Verify your email address by clicking on the link you recieved in your email inbox.

=> Now click on login & login with your sandbox credentials to your sandbox account

=> After login in you can see the page as below, don't panic if the UI is changed, its the new UI as of Jan 2019, UI changes won't affect its primary features. 02.png


=> Click on settings icon, and from top menu select API. 03.png

=> Here you can generate your new sandbox api credentials. Lets go forward and create one. Click on Generate New Api Key. 

=> A new public & private key will have got created.

=> Lets go ahead and download the php sdk using github by downloading as ZIP.

=> Browse the documentation and click on get started -> Simple Server (from left menu) (https://developers.braintreepayments.com/start/overview)

=> From Set Up Your Server  dropdown select Php

=> Click on Download Php library or download it using composer
